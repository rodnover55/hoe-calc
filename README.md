# Калькулятор урона — Heroes of Might and Magic: Olden Era

Веб-приложение для расчёта урона в бою: слева характеристики атакующего,
в центре параметры самой атаки (удача, дистанция, модификаторы), справа —
защищающегося.

Характеристики можно заполнить вручную или выбрать юнита из пресетов:
фракция (Храм, Некрополис, Сильван, Подземелье, Улей, Раскол, Нейтралы) →
юнит → грейд (база или одно из двух альтернативных улучшений). При выборе
подставляются статы юнита и показывается его портрет.

## Формула

```
урон = кол-во × урон существа × (20 + ATK) / (20 + DEF)
       × общие модификаторы × типовые модификаторы (мин. 10%) × удача
       × дальность × режим
```

- Удачный удар ×1.5, неудачный ×0.5.
- Итог всегда наносит минимум 1 урона.

Режим атаки выбирается кнопками и строится по способностям выбранного
юнита (карта слагов — в `src/abilityEffects.ts`):

- Стрелок (`ranged_attack`): дальняя атака теряет 10% за каждый гекс сверх
  трёх (максимум −50%); вплотную бьёт с половинным уроном. «Снайпер»
  (`sharpshooter`) стреляет без штрафа дальности, «Дуэлянт» (`duelist`)
  бьёт вплотную без штрафа.
- Боевые стойки (`fighting_style_*`) и «Пронзающий выстрел»
  (`piercing_shot`) — альтернативные атаки с множителем ×0.5.
- Активные способности с собственным уроном («Взор Бездны», «Ледяное
  дыхание», «Звездопад», «Мгновенная кара» и другие) — отдельные режимы:
  урон считается по формуле из описания способности (доля урона обычной
  атаки или `база + множитель × численность`), без АТК/ЗЩТ,
  модификаторов, дальности и удачи. Магический урон снижает «Защита от
  магии» цели, чистый не снижается ничем.
- Постоянные защитные способности цели — «Защита от выстрелов»
  (`ranged_defence_*`), «Защита в ближнем бою» (`melee_defence_*`),
  «Презрение» (`disdain`) — автоматически уменьшают урон
  соответствующего типа атаки (учитываются как типовые модификаторы).
- «Драконий клинок» (`draconic_blade`) удваивает урон ближней атаки.
- Ответный удар проставляется автоматически: дальняя атака, атака через
  гекс (`long_reach`, стойка-плевок `fighting_style_viscous_spit`),
  активные способности и «Стремительный удар» (`swift_strike`) ответа не
  провоцируют; галочку можно переключить вручную.
- «Двойной удар» (`double_strike`) и «Двойной выстрел» (`double_shot`,
  только при стрельбе) добавляют второй удар: он наносится после ответа
  выжившими атакующими по остатку отряда защитника. Удача в калькуляторе
  действует на оба удара одинаково — в игре она выпадает на каждый удар
  отдельно. На активные способности двойной удар не распространяется.

Прочие эффекты способностей (дебаффы и DoT от атак, площадь стоек и
мультицельных атак, ауры, временные баффы, условные бонусы вроде «+50%
урона целям выше рангом») в расчёт не входят — способности юнита
показываются списком в его карточке.

Данные — официальная вики игры; игра в раннем доступе, цифры могут меняться.

## База юнитов

Каждый юнит — YAML-файл в `src/data/units/<фракция>/<slug>.yaml`:

```yaml
id: warden_gnat
name: "Страж-мошка"
nameEn: "Warden Gnat"
faction: hive
tier: 1
grade: 1            # 0 — база, 1–2 — альтернативные улучшения
upgradeOf: gnat     # только у улучшений
image: units/hive/warden_gnat.webp
stats:
  health: 9
  attack: 3
  defense: 6
  damageMin: 2
  damageMax: 4
  initiative: 5
  speed: 5
attackType: long_reach   # melee | long_reach | ranged
flying: false
abilities:               # все способности юнита с olden-era.com
  - id: long_reach       # слаг — имя файла иконки на сайте
    name: "Атака на расстоянии"
    description: "Атакует отряд противника через один гекс. Не провоцирует контратаку и не занимает гекс рядом с целью."
growth: 6
cost: 24
source: https://www.olden-era.com/ru/units/warden_gnat
```

Портреты лежат в `public/units/<фракция>/`. Файлы загружаются в приложение
через `import.meta.glob` в `src/units.ts`.

Данные и картинки собраны с [olden-era.com](https://www.olden-era.com)
(фанатская база по игре). Обновить базу целиком:

```bash
npm run update-units
```

## Разработка

```bash
npm install
npm run dev      # dev-сервер с HMR
npm run build    # проверка типов + продакшен-сборка в dist/
npm run preview  # локальный просмотр собранного приложения
```

Стек: React 19, TypeScript, Vite. Логика расчёта — в `src/formula.ts`,
карта способностей и режимов атаки — в `src/abilityEffects.ts`,
интерфейс — в `src/App.tsx`.
