# Калькулятор урона — Heroes of Might and Magic: Olden Era

Веб-приложение для расчёта урона в бою: слева характеристики атакующего,
в центре параметры самой атаки (удача, дистанция, модификаторы), справа —
защищающегося.

## Формула

```
урон = кол-во × урон существа × (20 + ATK) / (20 + DEF)
       × общие модификаторы × типовые модификаторы (мин. 10%) × удача × дальность
```

- Удачный удар ×1.5, неудачный ×0.5.
- Дальнобойная атака теряет 10% за каждый гекс сверх трёх (максимум −50%).
- Стрелок вплотную или выстрел через препятствие — половинный урон.
- Итог всегда наносит минимум 1 урона.

Данные — официальная вики игры; игра в раннем доступе, цифры могут меняться.

## Разработка

```bash
npm install
npm run dev      # dev-сервер с HMR
npm run build    # проверка типов + продакшен-сборка в dist/
npm run preview  # локальный просмотр собранного приложения
```

Стек: React 19, TypeScript, Vite. Логика расчёта — в `src/formula.ts`,
интерфейс — в `src/App.tsx`.
